# 🎬 Movie Listing App

A modern movie discovery application built with React, Redux Toolkit, and Material UI.

Users can browse movies, search titles, filter by genres, and explore movie details through an elegant card-based interface.

---

## ✨ Features

✔️ Browse movie collection

✔️ Search movies by title

✔️ Filter movies by genre

✔️ Responsive movie cards

✔️ Material UI components

✔️ Redux Toolkit state management

✔️ Dynamic dropdown filters

✔️ Rating display

✔️ Fixed navigation bar

✔️ Modern and clean UI

---

## 🛠️ Tech Stack

### Frontend

- React.js

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Move into the project

```bash
cd movies-app
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm start
```

Application runs at

```text
http://localhost:3000
```
